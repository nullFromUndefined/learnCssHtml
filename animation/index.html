Шпаргалка

С помощью ключевых кадров @keyframes делаем взаимодействие с страницей более динамичным.

/* селектор */

.animation__circle{

width: 100px; height: 100px; border-radius: 50%; background-color: #5e5373; position: relative; animation-name: circle; animation-duration: 1s; animation-timing-function: linear; animation-iteration-count: 3;

}

/* ключевые кадры */

@keyframes circle{

0%{ left: 0%; } 50%{ border-radius: 0%; } 100%{ left: 50%; }

}

ключевые свойства для запуска анимации:

Все эти стили указываем на элемента который необходимо анимировать.

animation-name: white-circle; - указываем в классе какая анимация будет использоваться

animation-duration: 2s; - для запуска анимации необходимо задать время которое она будет выполняться.

Дополнительные совойства

animation-play-state: paused; - ставим анимацию на паузу, например при наведении мышью (псевдокласс hover)

animation-delay: 1s; - отвечает за задержку перед началом анимации

animation-iteration-count: infinite; - указываем сколько раз анимация должна повторяться. infinite - бесконечно, можно указать число - количество повторений.

animation-fill-mode: none; - Определяет какие значения анимируемых css свойств применятся к объекту после завершения анимации
none - значение по умолчанию
forwards - применяются конечные стили

Универсальное задание параметров для анимации

animation: 3s linear 1s infinite running slidein; - краткая форма
animation это короткая запись (последовательность указания параметров как указано дальше) для animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, animation-direction, animation-fill-mode и animation-play-state.

animation: circle 1s linear infinite normal running 0s both;

Если нужно применить сразу 2 анимации к одному элементу, то просто в свойстве animation через запятую перечисляем вторые параметры:
animation: circle 1s linear infinite normal running 0s both,
red-circle 5s linear infinite normal running 0s both;
